Estadística y Datamining
Objetivo del módulo: antes de abordar cualquier problema de Machine Learning es obligatorio mirar a los datos, ver qué podemos preguntar a los datos y qué información podemos sacar de ellos. El análisis exploratorio de datos es el primer paso que todo Data Scientist tiene que tomar. Para ello es fundamental tener un buen conocimiento de estadística que permite saber si ciertas variables tienen relación o no, o si varios grupos de datos se pueden considerar diferentes o iguales. Una vez realizado el análisis exploratorio, se pueden realizar los primeros modelos predictivos mediante diferentes técnicas de datamining.
Módulo: a distancia + práctica
Duración: 32 horas
Lenguajes: R
Herramientas: Rstudio
